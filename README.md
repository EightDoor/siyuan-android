## 搭建步骤

1. 到 https://github.com/siyuan-note/kernel-libs 下载并拷贝内核库 app/libs/kernel.aar
2. 从桌面端安装路径 `resources` 下拷贝资源文件并打包 app/src/main/assets/app.zip
   * appearance
   * guide
   * stage
